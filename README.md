# tomori
```tomori``` is tools for analyze Drive-by Download

## Require
- PHP 7
- SQLite 3
- Composer

## Install
```sh
# clone repository
$ git clone https://github.com/koike/tomori.git

# setup files & create db
$ php install.php

# write api key
$ vi .env
```

## Usage
```sh
# start
$ php tomori.php
```
