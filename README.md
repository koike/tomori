# tomori
```tomori``` is tools for analyze Drive-by Download on Twitter

## Require
- PHP ~7.0
- SQLite 3
- Composer

## Usage
```sh
# clone repository
$ git clone https://github.com/koike/tomori.git

# setup files & create db
$ php install.php

# write api key
$ vi .env

# start
$ php tomori.php
```
